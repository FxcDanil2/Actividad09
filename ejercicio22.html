Resumen de Propiedades

Propiedad Descripción
border-top Define el borde superior de un elemento
border-bottom Define el borde inferior de un elemento
margin Define el espacio exterior alrededor de un elemento
padding Define el espacio interior dentro de un elemento
vertical-align Alinea elementos verticalmente respecto a su línea base
text-indent Define la sangría de la primera línea de texto
text-wrap Controla cómo se ajusta el texto dentro de su contenedor
text-stroke Define el contorno del texto (requiere prefijo -webkit-)
